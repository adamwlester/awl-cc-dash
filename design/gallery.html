Component Gallery — States Catalog

The visual component catalog for the AWL Multi-Agent Dashboard design system. Every reusable component and basic building block appears here, grouped by its canonical name, shown once per visual state — each state labeled with its precise class combo. This page renders states, not behaviors — it exists because the running app can't show everything at once (most badge variants, all five Inbox card types, and every hover/focus/open state are one data-state away from being invisible). It links the same tokens.css and styles.css the mockup uses, so specimens render identically to the live components; only page-layout scaffolding (the gx-* classes) is added here — including a gallery-only gx-force-* set that mirrors styles.css's :hover / :focus / :disabled deltas so a static specimen can show those interaction states.
