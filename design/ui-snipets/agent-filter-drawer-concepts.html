Prompts · agent-selector as a side drawer

The persistent Source + Target column eats ~164px of the Prompts panel (and the same list is the Feed → Inbox filter). Three low-styling ways to tuck it away so Compose gets the full width — each reuses the existing identity rows, the Source dropdown, the All/None mini-link, and the --select selection fill. Try the controls. Whatever wins applies 1:1 to the Inbox filter.

Option 1 · Slide-over drawer

How: Compose is full-width; a slim To bar shows the current targets as tiles. Click To/Edit → an overlay drawer slides in with Source + Target. + Most space saved; targets still glanceable; drawer can hold more later. − An overlay (briefly covers compose); needs a scrim + open/close state.
