Component Gallery

The visual component catalog for the AWL Multi-Agent Dashboard design system. Every component in the registry appears here exactly once, grouped by its canonical name, each shown with the rendered specimen, a readable name, its literal data-comp kebab slug, and a one-line purpose blurb. This page renders states, not behaviors — it exists because the running app can't show everything at once (most badge variants and all five Inbox card types are one data-state away from being invisible). It links the same tokens.css and styles.css the mockup uses, so specimens render identically to the live components; only page-layout scaffolding (the gx-* classes) is added here.
