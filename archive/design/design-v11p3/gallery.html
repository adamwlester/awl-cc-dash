Component Gallery — Interactive Catalog

Every reusable component and basic building block in the design system. This page shares tokens.css, styles.css and behavior.js with the mockup, so each operable specimen is the genuine component driven by the genuine interaction code — never a frozen copy.

The operable specimens are live: hover, click, open their menus. Components with several looks (button tiers, badge colours) show every variant side by side; controls with menus or toggles show one live instance you can drive. Only page scaffolding (the gx-* classes) is added here.
