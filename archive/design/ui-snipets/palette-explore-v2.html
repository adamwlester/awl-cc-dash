Explore — agent identity family (16 colors)

Each agent owns one color, used everywhere it appears (graph dot + tile, feed name, rows, requests). The goal: a set that's internally even (no agent louder than another) and sits in a different register from the UI signals above, so a colored tile never gets mistaken for pink-selected, teal-secondary, or a green/amber/red state. All three families are built in OKLCH — 16 hues spaced evenly around the wheel at a fixed lightness; only the chroma/lightness "quality" changes per family.
